Using Animate.css in Vue projects

Useful links:

animate.css

Process

  • install animate.css using npm

    npm install animate.css --save

  • if successful, you may see:

    "animate.css": "^4.1.1", in the package.json» "dependencies"

    • note: may it is not version 4.1.1

  • using animate in the Vue project

    • in main.js,
      • import it using import animated from 'animate.css'
      • using it by Vue.use(animated)

  • then you can use it

    <div class="animate__animated animate__fadeInDown" > </div>
    • note: you have to add class animate__animated
    • note: then you need to add class animate__X , where X is based on the feature you want, reference to this link, you can view different demo of animations.
