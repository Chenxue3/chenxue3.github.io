Set up an Electron Project with VUE

Jan 04, 2022
1 minute read

Download Resources

  • Node.js

    Node.js download

    using

    node -v
npm -v

    to check.

  • Vue

    initial Vue by using the command:

    npm install -g @vue/cli
# OR
yarn global add @vue/cli

    using

    vue --version

    to check

Create a new Vue project

  • create
vue create [vue-electorn-demo-name]
  • run the project
npm run serve

Add electron

Using vue-cli-plugin-electron-builder

vue add electron-builder

Run Project in Electron

 npm run eletron:serve

the project will look like:

vue electron
